Aplikasi Pemesanan Baju Bola Keluarga

Deskripsi Singkat:
Aplikasi ini digunakan untuk memesan baju bola untuk pemesan utama dan seluruh anggota keluarganya. Pemesan mengisi data pribadi, memilih nomor punggung (sama untuk semua anggota), lalu setiap anggota keluarga bisa memilih ukuran baju masing-masing. Harga dihitung otomatis sesuai ukuran.

Fitur Utama

Data Pemesan Utama

Input Nama Pemesan (text)

Input Nomor WhatsApp (number/text)

Input Nomor Punggung (number) → nomor ini berlaku untuk semua baju keluarga

Detail Pemesanan Keluarga

Input jumlah anggota keluarga (termasuk pemesan utama)

Untuk setiap anggota, data yang diinput:

Nama Anggota (text)

Umur (number)

Jenis Kelamin (radio: Laki-laki / Perempuan)

Ukuran Baju (dropdown) + Harga otomatis tampil:

Anak Kecil (S Anak) → Rp80.000

Anak Sedang (M Anak) → Rp80.000

Anak Besar (L Anak) → Rp80.000

Dewasa S → Rp100.000

Dewasa M → Rp100.000

Dewasa L → Rp100.000

Dewasa XL → Rp110.000

Dewasa XXL → Rp110.000

Dewasa XXXL → Rp120.000

Harga Baju (auto calculate sesuai ukuran)

Catatan: Nomor punggung otomatis sama dengan nomor pemesan utama

Ringkasan Pesanan

Menampilkan daftar semua anggota + ukuran baju + harga baju + nomor punggung

Menampilkan Total Pesanan (jumlah baju)

Menampilkan Total Harga (Rp)

Konfirmasi Pesanan

Checkbox: “Ya, data sudah benar”

Tombol: Submit

Output / Data yang Disimpan

Nama Pemesan

Nomor WhatsApp

Nomor Punggung

List Anggota Keluarga (Nama, Umur, Jenis Kelamin, Ukuran Baju, Harga Baju, Nomor Punggung)

Total Pesanan

Total Harga
